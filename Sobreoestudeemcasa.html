Estude em casa - Online

Esta página web foi pensada, a esses alunos que estão buscando informações pela Web e não estão consegundo se organizar e filtrar essas informações, essa página web vem com o objetivo de juntar essas informações e oferecer de forma simplificada, apenas acessando a página web “Estude em casa – Online” que contém links de sites que foram selecionados de forma criteriosa e detalhada para o estudante..

Como dela foi desenvolvida?

Para construir o corpo do trabalho, foi utilizado a linguagem HTML e CSS para definir as informações e formatações do site, e seguindo ao método padrão, foi iniciado com programação Html, agrupando links que vão dar acesso a outras páginas, que foram selecionadas com muito critério para repassar aos visitantes.
